# Thrusters
- When sending serial signals, you will reference the corrosponding thruster
  in the format `[position][left/right]`
- example: to access the front right thruster, you would use `fr`, middle left would be `ml`
|          | [l]eft | [r]ight |
|----------|--------|---------|
| [f]ront  | fl     | fr      |
| [m]iddle | ml     | mr      |
| [b]ack)  | bl     | br      |
